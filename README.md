# rakitpc
Website interaktif untuk merakit PC secara step-by-step, menampilkan pilihan CPU, motherboard, RAM, GPU, PSU, SSD, dan casing lengkap dengan spesifikasi, harga, dan link toko di Indonesia. Cocok untuk pemula maupun pecinta rakit PC yang ingin membandingkan komponen dan memastikan kompatibilitas sebelum membeli.
